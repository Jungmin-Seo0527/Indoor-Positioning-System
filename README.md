# Indoor Positioning System

## 프로젝트 개요

* 비콘에서 송신하는 RSSI값을 이용해서 실내 측위를 하는 시스템
* RSSI의 낮은 신뢰성, 방향성이 없는 단점을 딥러닝을 적용하여 개선

## 사용 기술

* python 3.7
* tensorflow 1.3

## 프로젝트 실행

* `Main.py` 실행
